1. Overview

Molvicos OS is a browser-based AI operating system that brings together 18 AI-powered applications, an intelligent assistant (MIRA), 11 external service extensions, and 6 AI providers — all in a desktop-like interface accessible from any browser.

Key highlights:

  • 18 apps for content creation, SEO, freelancing, email, strategy, and automation
  • MIRA — your intelligent OS assistant that can control the system and answer questions
  • 6 cloud AI providers + local models via Ollama (zero cost, full privacy)
  • 11 extensions to connect GitHub, Cloudflare, Stripe, Telegram, Notion, and more
  • 5 themes, 5 languages (EN, ES, DE, FR, ZH)
  • Desktop drag & drop, resizable windows, Dock, Command Palette

2. Getting Started

Onboarding (5 steps)

  1. Language — Choose your preferred language (EN, ES, DE, FR, ZH)
  2. Hardware Scan — Automatic detection of CPU, RAM, and GPU to recommend optimal AI models
  3. API Keys — Connect at least one AI provider. Groq is recommended (free tier, ultra-fast)
  4. Local Models — Optionally install Ollama for private, zero-cost AI inference
  5. Tutorial — Interactive guided tour of the OS interface
💡 You can re-open the tutorial anytime from Settings → General → Open Tutorial

3. Desktop & Window Management

Desktop

  • Double-click any app icon to open it
  • Drag icons to rearrange them (positions are saved automatically)
  • Right-click context available on desktop items

Windows

  • Drag the title bar to move a window
  • Resize from edges and corners
  • Click a window to bring it to front
  • Minimize — Hides window (click Dock icon to restore)
  • Maximize — Fills the screen
  • Close — Closes the app window

Dock

  • Fixed at the bottom of the screen
  • Click an app to open/focus it; click again to minimize
  • Active apps show a dot indicator
  • Customize which apps appear in the Dock from Settings → Dock

Top Bar

  • Left: MolvicOS logo
  • Center: Clock
  • Right: Language selector, credit counter, theme toggle, account menu

5. MIRA — Molvicos Intelligent Research Assistant

MIRA is your built-in AI assistant accessible from the floating button at the bottom-right corner of the OS.

What MIRA Can Do

  • OS Control — Open/close apps, change theme, switch language (via tool calls)
  • Research — Answer questions, explain concepts, help with any task
  • App Guidance — Explain how to use any Molvicos OS feature
  • Extension Actions — Execute actions on connected services (create GitHub repos, send Telegram messages, query Supabase, etc.)
  • Proactive Suggestions — Context-aware suggestion chips that update every 30 seconds

MIRA Core Tools

ToolDescription
open_appOpen any application by its ID
close_appClose an open application
change_themeSwitch theme (noir, icaro, synthwave, ocean, matrix)
suggest_promptSuggest a prompt for Prompt Lab
change_languageChange OS language (en, es, de, fr, zh)
report_feedbackOpen the feedback/bug report form

MIRA Provider Configuration

MIRA uses a separate model from the main apps. Go to Settings → AI & Models to select a specific provider for MIRA, or leave it on "Auto" to use the first available key.

Auto-detection priority: Groq → Gemini → Mistral → OpenAI → Anthropic → GitHub Models

Chat Limits

  • Context window: last 20 messages
  • Maximum stored messages: 100 (a warning appears when approaching the limit)
  • Each MIRA message costs 1 credit

14. Data Storage

All data is stored locally in your browser:

  • localStorage — Settings, API keys, credits, usage logs, app history, Dock configuration, extension configs
  • IndexedDB — Prompts, conversations, files (database: molvicos_db)
⚠️ API keys never leave your browser. They are sent only to Molvicos server-side routes which proxy requests to AI providers. Keys are never exposed to third-party client-side code.

15. FAQ

Q: Do I need to pay to use Molvicos OS?

No. The free plan includes 50 credits/month and access to core features. During the beta period, credits are virtually unlimited.

Q: Which AI provider should I choose?

Groq is recommended — it offers a generous free tier with ultra-fast inference. Gemini and Mistral also have free tiers. You only need one provider.

Q: Can I use Molvicos OS offline?

Partially. The PWA works offline for the interface, and local models via Ollama work without internet. Cloud AI features require an internet connection.

Q: Where is my data stored?

Everything stays in your browser (localStorage + IndexedDB). Nothing is sent to external servers except AI inference requests which go through Molvicos server-side proxies.

Q: How do I get more credits?

Credits reset monthly. Upgrade to Pro for unlimited credits, or use local models (Ollama) which cost 0 credits.

Q: How do I connect extensions?

Open the Extensions app, enable the desired extension, enter the required API key/token, and save. MIRA will then be able to use those extension tools.

Q: Can MIRA control my GitHub/Cloudflare/etc.?

Yes, once you configure the corresponding extension, MIRA can execute actions via tool calls (e.g., create a repo, list deployments, send a Telegram message).
